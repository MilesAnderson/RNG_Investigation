# RNG Investigation
This project implements four different random number generators (RNG), runs them against three RNG tests, and discusses the results in a research paper.

The four RNGs implemented were:
- Algorithm 1: Park-Miller Linear Congruential Generator
- Algorithm 2: Middle-Square Generator
- Algorithm 3: XORShift Generator
- Algorithm 4: HMAC-DRBG Generator

The three RNG tests implemented were:
- Frequency Test
- Runs Test
- Serial Test

In addition to being run against the RNG tests the runtime of each algorithm was found. The RNGs were tested and timed in main.py.

## Paper

The full discussion and analysis of the results can be found in **RNG_Investigation.pdf**.
